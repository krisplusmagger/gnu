18static const char* __doc_gr_qtgui_time_raster_sink_f = R"doc(A graphical sink to display multiple signals on a time_raster plot.
19
20This is a QT-based graphical sink that takes set of a floating point streams and plots a time_raster (spectrogram) plot.
21
22
23The sink supports plotting streaming float data or messages. The message port is named \"in\". The two modes cannot be used simultaneously, and should be set to 0 when using the message mode. GRC handles this issue by providing the \"Float Message\" type that removes the streaming port(s).
24
25
26This sink can plot messages that contain either uniform vectors of float 32 values (pmt::is_f32vector) or PDUs where the data is a uniform vector of float 32 values.
27
28Constructor Specific Documentation:
29
30Build a floating point time raster sink.
31
32Args:
33 samp_rate : sample rate of signal
34 cols : number of cols to plot
35 rows : number of rows to plot
36 mult : vector of floats as a scaling multiplier for each input stream
37 offset : vector of floats as an offset for each input stream
38 name : title for the plot
39 nconnections : number of streams connected
40 parent : a QWidget parent object, if any)doc";
