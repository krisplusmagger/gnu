const char* __doc_gr_blocks_selector
static
Initial value:
= R"doc(output[output_index][i] = input[input_index][i]
Connect the sink at input index to the source at output index.
Samples from other input ports are consumed and dumped at a rate which is limited by (but not necessarily the same as) the rate of samples copied from the active input to the active output.
Other output ports produce no samples.
Constructor Specific Documentation:
Create new selector block and return a shared pointer to it
Args:
itemsize : size of the input and output items
input_index : the initially active input index
output_index : the initially active output index)doc"
